API Overview

This API provides backend functionality for various document processing tasks as part of American Power Marketing Corp.'s digital tools. Below are the available endpoints and how to use them.

YouTube Downloader

Download YouTube videos or audio in different formats.

GET /youtube/download?url=YOUTUBE_URL&format=mp3|mp4

Parameters:

url (required) - The YouTube video URL to download
format (optional) - Either "mp3" for audio only or "mp4" for video (default: mp4)

Example:

GET https://document-editor-tool.onrender.com/youtube/download?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ&format=mp3

PDF Merger

Merge multiple PDF files into a single document.

POST /pdf/merge

Usage:

Send a POST request with multiple PDF files in the "files" field. The API will return the merged PDF.

Example (cURL):

curl -X POST -F "files=@file1.pdf" -F "files=@file2.pdf" https://document-editor-tool.onrender.com/pdf/merge --output merged.pdf

Document Converter

Convert documents between different formats (Word, Excel, PDF, etc.).

POST /convert

Usage:

Send a POST request with a file in the "file" field. The API will return the converted document.

Supported Formats:

PDF, Word (.doc, .docx), Excel (.xls, .xlsx), PowerPoint (.ppt, .pptx), images (.jpg, .png), and text files.

Example (cURL):

curl -X POST -F "file=@document.docx" https://document-editor-tool.onrender.com/convert --output converted.pdf
